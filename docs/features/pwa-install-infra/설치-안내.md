| 항목                 | 상태                                  |
| -------------------- | ------------------------------------- |

## 설치하는 법 (2단계)

### 1단계. 주소를 폰에서 열리게 만들기

PWA가 들어있는 주소:

```
https://sdg-frontend-a9fmaryhw-a1rhuns-projects.vercel.app
```

이 주소가 지금 Vercel 로그인 벽에 막혀 있음. 푸는 방법 둘 중 하나:

**방법 A — 공개로 바꾸기 (남에게도 시연 가능)**

1. `vercel.com` 로그인 (본인 계정)
2. `sdg-frontend` 프로젝트 → **Settings**
3. **Deployment Protection** 메뉴
4. **Vercel Authentication** → 끄기, 또는 범위를 **"Only Production Deployments"**로
5. **Save**

> 주의: preview가 인터넷에 공개됨. mock 데이터뿐이라 민감정보는 없음.
> 데모 끝나면 다시 켜도 됨.

**방법 B — 나만 보기 (설정 변경 없음)**

- 폰 브라우저로 위 주소 열기 → Vercel 로그인 화면 뜨면 본인 계정으로 로그인
- 로그인 후 그 브라우저에서 정상 열림 (단, 본인만 가능)

### 2단계. 폰에서 설치

**📱 Android (크롬)**

1. 크롬으로 주소 열기
2. 상단 녹색 **"설치" 버튼** 또는 크롬 **⋮ → "앱 설치 / 홈 화면에 추가"**
3. 확인 → 홈 화면에 아이콘 생성

**📱 iPhone (사파리 — 크롬 아님!)**

1. **사파리**로 주소 열기
2. 하단 **공유 버튼** (□↑)
3. **"홈 화면에 추가"**
4. 추가 → 홈 화면에 아이콘 생성

설치 후 아이콘 탭 → **주소창 없는 전체화면 앱**으로 실행됨.

## 설치하면 뭐가 되나

**✅ 되는 것 (앱 경험)**

- 홈 화면 아이콘 (지금은 녹색 임시 로고)
- 전체화면 실행 (브라우저 티 안 남)
- 앱 전환기에 독립 앱으로 표시
- 아이콘 길게 누르면 바로가기 (진단 / 챗봇 / 캘린더)

**🟡 아직 안 되는 것 (백엔드 필요)**

- 실제 사진 → AI 진단
- 진짜 진단 기록 / 챗봇 / 캘린더 데이터
- 오프라인 캐싱 (mock 모드에선 꺼짐)

## 남은 작업 (이 다음에 할 것들)

| 작업                          | 막힌 이유                  |
| ----------------------------- | -------------------------- |
| 실제 로고 아이콘 교체         | 디자인 자산 필요           |
| 실기기 설치 / Lighthouse 점검 | 위 1단계(주소 공개)면 가능 |
| API 캐싱·오프라인 데이터      | 백엔드 API 확정 후         |
| 앱스토어 배포 (Capacitor)     | 추후 단계                  |

---

## 용어 한 줄 정리

- **PWA** = 브라우저에서 "설치"하는 앱. 스토어 없이 홈 화면에 깔림.
- **스토어 앱(나중)** = PWA를 Capacitor로 감싸 Play스토어/App스토어에 올리는 것.
- **mock 데이터** = 백엔드 대신 임시로 넣어둔 가짜 데이터.
